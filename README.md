Go Jet support for intellij
=====

This intellij plugin adds support for the Jet template engine http://github.com/CloudyKit/jet

 - [x] Parser
 - [x] Lexer
 - [x] Multilayer Syntax highlighter
 - [ ] Formatting
 - [ ] Auto-completion
 - [ ] Write tests
